# Documentación de módulos integrados

Proyecto: Metrónomo Online

## Módulos integrados

- Registro de usuarios
- Inicio de sesión
- Panel principal
- Configuración del metrónomo
- Guardado de preferencias
- Cierre de sesión
- Base de datos MySQL

## Ambiente de desarrollo

- XAMPP
- Apache
- MySQL/MariaDB
- PHP 8
- Visual Studio Code
- GitHub

http://localhost/Proyecto_Metronomo_Online/
